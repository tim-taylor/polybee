# polybee
An agent-based model of bee movement in polytunnels

## Requirements

### Boost
Either:
* `libboost-dev`

Or just the components we actually use:

* `libboost-program-options-dev`
* `libboost-serialization-dev`

### OpenCV
* `libopencv-dev`
